Introduction

Multiverse helps companies set up and hire apprentices for modern-day roles. We create apprenticeship opportunities as an alternative to university and match apprentices with employers.

My role
Build Multiverse a unified design system.

The business goal
Harmonize the experience across their five different web applications and enable the company to scale quickly outside of London.

Impact:
To date, 4/5 applications are using the design system, and we've expanded outside of London and to the US, in other words, its working.

Component Guidelines

Each component comes with a set of rules that help the design team create consistantly great user journeys.

AA accessible

Our design principle 'Empower' guided us to bake WCAG AA guidelines into our system components.

Consistant Code

We created a single place for all developers to copy code for each component so that live code is consistant and free of errors.
